Download KerfDesk Preview

KerfDesk Preview is free, public, and offline-capable. These early desktop builds are unsigned. All three buttons below open the same public KerfDesk GitHub releases page; choose the newest release marked Pre-release, then select the exact asset for your computer.

Preview updates are manual. KerfDesk may notify you that a newer Preview exists, but it never downloads or installs an unsigned update. Finish any machine work, close KerfDesk, return to this page, and install the newer exact-version asset yourself.

In each filename, <version> is the prerelease tag without its leading v (for example, tag v0.2.0-preview.1 uses version 0.2.0-preview.1).

Install an unsigned Preview

Windows SmartScreen

  1. Run the downloaded setup file.
  2. Only continue if it came from the public repository linked above and its filename matches the Windows pattern.
  3. If SmartScreen shows Windows protected your PC, select More info.
  4. Confirm the publisher is shown as unknown, then select Run anyway.

macOS Gatekeeper

  1. Open the DMG and drag KerfDesk.app to Applications.
  2. Control-click KerfDesk.app in Applications, choose Open, then choose Open again.
  3. If macOS still blocks it, open System Settings > Privacy & Security and choose Open Anyway. On macOS 12 Monterey, use System Preferences > Security & Privacy > General > Open Anyway.

Mac Preview builds are both unsigned and unnotarized. KerfDesk does not disable or silently bypass Gatekeeper.

Linux: there is no Linux desktop Preview yet. Use the KerfDesk web app in Chrome or Edge; it has the same CAM features and can be installed from the browser.
