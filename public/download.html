Download KerfDesk for Windows

Windows 10 or 11, 64-bit. Free and offline-capable.

Download the installer (.exe)

The desktop app is the same KerfDesk you use in the browser, in a native window with a native menu and robust offline support. It connects to your GRBL laser or CNC over USB. Updates are installed manually until code signing is enabled.

First launch: a Windows warning is expected. This early build is not yet code-signed, so Microsoft Defender SmartScreen may show “Windows protected your PC.” Click More info → Run anyway to install. A signed build (no warning) is coming.
On macOS or Linux? There's no desktop installer yet — use the web app at kerfdesk.com in Chrome or Edge (same features, installable from the browser). Open KerfDesk in this browser →
